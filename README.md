# Usage
Restore package with:
```
composer install
```

Remplace variables in config.php file and upload to server.

```php
$BUCKET_NAME = '';

// NOTE: to create private key JSON file: 
// https://console.cloud.google.com/iam-admin/serviceaccounts?project=  
$privateKeyFileContent = '';
```
